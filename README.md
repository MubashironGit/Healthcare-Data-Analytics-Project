# Healthcare Data Analytics Project

## Tools Used
- MySQL
- Microsoft Excel
- Power BI
- Tableau

## Project Overview
This project analyzes healthcare data to extract insights related to patient trends, operational efficiency, and performance metrics.

## Files Included
- *MySQL:* SQL queries used for analysis
- *Excel:* Raw and cleaned datasets
- *Power BI:* Interactive dashboards (zipped)
- *Tableau:* Visualization workbook

## Key Insights
- Trend analysis of patient data
- KPI dashboards for healthcare performance
- Data-driven insights for decision-making
